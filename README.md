# CS-405-Secure-Coding

This repository contains coursework completed for **CS 405: Secure Coding**. The projects demonstrate the application of secure software development principles, secure coding standards, risk assessment, encryption, static analysis, testing, and security policy development. Throughout the course, emphasis was placed on building security into the software development lifecycle using industry best practices and a defense-in-depth approach.

---

## Repository Contents

### Project One – Green Pace Security Policy
A comprehensive security policy developed for the fictional company Green Pace. The policy includes:

- Ten secure coding standards based on the SEI CERT C++ Coding Standard
- Risk assessments for each coding standard
- Automated detection tools and DevSecOps integration
- Encryption policies
- Authentication, Authorization, and Accounting (AAA) policies
- Defense-in-depth strategy and security recommendations

**File:** `Project One Security Policy.docx`
